# Submission Checklist

## Product readiness

- [ ] Deployed Vercel URL works.
- [ ] `OPENAI_API_KEY` is set in Vercel environment variables.
- [ ] Chatbot responds with short, human guidance.
- [ ] Guided check reaches results quickly.
- [ ] Results cards show documents and next steps directly.
- [ ] Results cards only show `Open official program` and `Open Google Maps search`.
- [ ] Human verification module does not show `Ask chatbot`.
- [ ] Disclaimer appears first and includes the ServiceBridge AI logo.
- [ ] Voice input works in a supported browser.
- [ ] Read-aloud works in a supported browser.

## Demo checks

- [ ] BEAM scenario tested.
- [ ] SNAP scenario tested.
- [ ] Medicaid / CHIP scenario tested.
- [ ] Zimbabwe National ID scenario tested.
- [ ] State ID / REAL ID scenario tested.
- [ ] Missing source URL does not show a broken button.
- [ ] Missing map/search data does not show a broken map button.
- [ ] Mobile layout is readable.

## Demo video structure

- [ ] 30 seconds: problem and who it affects.
- [ ] 60 seconds: what the AI does and how.
- [ ] 30 seconds: one moment where it makes a difference.
- [ ] 30 seconds: responsible AI choice and human role.
- [ ] 30 seconds: what to build next.

## Final submission assets

- [ ] GitHub repo link.
- [ ] Vercel deployed app link.
- [ ] Demo video link.
- [ ] Short pitch description.
- [ ] Responsible AI explanation.
- [ ] Team/member details.
